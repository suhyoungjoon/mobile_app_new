# 🧪 자동 기능 테스트 계획

## 📋 테스트 범위

### 기능별 테스트 계획

#### 기능 1: 로그인 ⏳ (진행 중)
- [x] 로그인 화면 표시
- [x] 로그인 정보 입력
- [x] 로그인 버튼 클릭
- [x] 로그인 성공 확인
- [x] 하자 목록 화면 이동 확인

#### 기능 2: 하자 등록 (대기)
- [ ] 하자 등록 화면 이동
- [ ] 위치 선택
- [ ] 세부공정 선택
- [ ] 하자 내용 입력
- [ ] 사진 업로드 (근거리/원거리)
- [ ] 하자 등록 완료 확인

#### 기능 3: 장비점검 - 열화상 (대기)
- [ ] 장비점검 탭 이동
- [ ] 열화상 탭 선택
- [ ] 열화상 사진 업로드
- [ ] 위치 및 메모 입력
- [ ] 저장 확인

#### 기능 4: 장비점검 - 공기질 (대기)
- [ ] 공기질 탭 선택
- [ ] TVOC, HCHO 값 입력
- [ ] 위치 및 메모 입력
- [ ] 저장 확인

#### 기능 5: 장비점검 - 라돈 (대기)
- [ ] 라돈 탭 선택
- [ ] 라돈 값 입력
- [ ] 위치 및 메모 입력
- [ ] 저장 확인

#### 기능 6: 장비점검 - 레벨기 (대기)
- [ ] 레벨기 탭 선택
- [ ] 좌/우 수치 입력
- [ ] 위치 및 메모 입력
- [ ] 저장 확인

#### 기능 7: 점검원 등록 (대기)
- [ ] 점검원 등록 탭 이동
- [ ] 등록 정보 입력
- [ ] 등록 신청 완료 확인

#### 기능 8: 보고서 생성 (대기)
- [ ] 보고서 미리보기 화면
- [ ] PDF 생성 확인
- [ ] 보고서 다운로드 확인

#### 기능 9: 관리자 기능 (대기)
- [ ] 관리자 로그인
- [ ] 대시보드 확인
- [ ] 사용자 관리 화면
- [ ] 점검원 관리 화면
- [ ] 하자 관리 화면

## 🎯 진행 방식

1. **기능 1 테스트 완료** → 스크린샷 확인 → 리뷰
2. **리뷰 완료 후** → 기능 2 진행
3. **각 기능별로 동일한 프로세스 반복**
