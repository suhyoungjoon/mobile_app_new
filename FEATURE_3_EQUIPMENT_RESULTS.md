## 🔧 발견 및 수정된 이슈

### 1. 탭 전환 문제
- **문제:** 탭 버튼 클릭 시 "Node is either not clickable or not an Element" 오류
- **해결:** JavaScript로 직접 `showEquipmentTab()` 함수 호출

### 2. 입력 필드 클릭 문제
- **문제:** 입력 필드 클릭 시 동일한 오류 발생
- **해결:** JavaScript로 직접 `value` 설정 및 `input` 이벤트 발생

### 3. 저장 버튼 클릭 문제
- **문제:** 저장 버튼 클릭 시 동일한 오류 발생
- **해결:** JavaScript로 직접 `saveEquipmentInspection()` 함수 호출

2. **대기 시간 최적화**
