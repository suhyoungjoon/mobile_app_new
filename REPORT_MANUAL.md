## 📊 보고서 기능 개요

InsightI의 보고서 기능은 등록한 하자와 장비 점검 결과를 종합하여 보고서로 확인하고 발송할 수 있는 기능입니다.

### 주요 기능

- ✅ **보고서 미리보기**: 등록한 하자 목록을 화면에서 미리 확인
- ✅ **SMS 발송**: 보고서를 SMS로 발송 (모의 모드)
- ⏳ **PDF 다운로드**: PDF 파일로 다운로드 (향후 구현 예정)

### 보고서에 포함되는 내용

- **하자 정보**
  - 위치, 세부공정, 하자 내용
  - 메모 (선택사항)
  - 사진 (근거리/원거리)
  
- **장비 점검 결과** (점검원 권한 사용자)
  - 열화상 점검
  - 공기질 측정 (TVOC, HCHO, CO₂)
  - 라돈 측정
  - 레벨기 측정

## 📱 보고서 미리보기

보고서 미리보기 기능을 사용하면 등록한 하자 목록을 화면에서 확인할 수 있습니다.

### 사용 방법

#### 1단계: 로그인

먼저 앱에 로그인합니다.

![로그인 화면](test-screenshots/report-preview/report-preview-01-after-login-2025-11-19T11-09-01.png)

*로그인 후 하자 목록 화면*

**로그인 정보 입력:**
- 아파트(단지)명
- 동, 호수
- 성명
- 전화번호

로그인 버튼을 클릭하면 하자 목록 화면으로 이동합니다.

---

#### 2단계: 보고서 탭 클릭

하단 탭바에서 **"보고서"** 탭을 클릭합니다.

![하자 목록 화면](test-screenshots/report-preview/report-preview-01-after-login-2025-11-19T11-09-01.png)

*하단 탭바에서 "보고서" 탭을 클릭합니다*

또는 하자 목록 화면에서 **"보고서 미리보기"** 버튼을 클릭할 수도 있습니다.

---

#### 3단계: 보고서 미리보기 확인

보고서 미리보기 화면이 표시됩니다.

![보고서 미리보기 화면](test-screenshots/report-preview/report-preview-02-report-screen-2025-11-19T11-09-08.png)

*보고서 미리보기 화면이 표시됩니다*

---

#### 4단계: 하자 목록 확인

등록한 하자 목록이 카드 형식으로 표시됩니다.

![보고서 내용](test-screenshots/report-preview/report-preview-03-report-content-2025-11-19T11-09-11.png)

*하자 카드 형식으로 하자 목록이 표시됩니다*

**하자 카드 정보:**
- **위치 / 세부공정**: 하자가 발생한 위치와 세부공정
- **하자 내용**: 하자 상세 내용
- **메모**: 등록 시 입력한 메모 (있는 경우)
- **사진**: 근거리/원거리 사진 (있는 경우)

---

### 하자가 없는 경우

등록된 하자가 없는 경우 다음과 같은 메시지가 표시됩니다.

![보고서 내용 없음](test-screenshots/report-preview/report-preview-03-report-content-2025-11-19T11-09-11.png)

*하자가 없는 경우 "등록된 하자가 없습니다" 메시지가 표시됩니다*

> **"등록된 하자가 없습니다."**

이 경우 먼저 하자를 등록한 후 보고서를 확인하세요.

---

## 📤 보고서 발송

보고서를 SMS로 발송할 수 있습니다. (현재 모의 모드)

### 사용 방법

#### 1단계: 보고서 미리보기 화면으로 이동

보고서 미리보기 화면으로 이동합니다. (위의 "보고서 미리보기" 섹션 참조)

---

#### 2단계: 문자 발송 버튼 클릭

보고서 미리보기 화면 하단의 **"문자 발송(모의)"** 버튼을 클릭합니다.

![보고서 발송 버튼](test-screenshots/report-preview/report-preview-05-report-final-2025-11-19T11-09-13.png)

*하단의 "문자 발송(모의)" 버튼을 클릭합니다*

---

#### 3단계: 전화번호 입력

전화번호 입력 창이 표시됩니다.

```
보고서를 받을 전화번호를 입력하세요 (예: 010-0000-0000)
```

**전화번호 형식:**
- 예: `010-1234-5678`
- 예: `01012345678` (하이픈 없이도 가능)

---

#### 4단계: 발송 확인

발송이 완료되면 성공 메시지가 표시됩니다.

> ✅ **"SMS로 보고서가 발송되었습니다"**

---

### 보고서 발송 주의사항

⚠️ **현재 모의 모드**

현재 보고서 발송 기능은 **모의 모드**로 작동합니다. 실제 SMS가 발송되지 않으며, 테스트 목적으로만 사용됩니다.

실제 SMS 발송 기능은 향후 업데이트에서 제공될 예정입니다.

## 🔧 문제 해결

### 보고서 미리보기가 표시되지 않을 때

**증상**: 보고서 탭을 클릭해도 화면이 표시되지 않음

**해결 방법:**

1. **케이스 확인**
   - 먼저 케이스가 생성되어 있는지 확인하세요
   - 케이스가 없으면 "먼저 케이스를 생성해 주세요" 메시지가 표시됩니다

2. **인터넷 연결 확인**
   - 인터넷 연결 상태를 확인하세요
   - Wi-Fi 또는 모바일 데이터가 활성화되어 있는지 확인

3. **페이지 새로고침**
   - 브라우저에서 페이지를 새로고침 (F5 또는 새로고침 버튼)
   - 또는 앱을 다시 시작

4. **로그인 상태 확인**
   - 로그인이 만료되었을 수 있습니다
   - 다시 로그인해 보세요

---

### 하자 목록이 표시되지 않을 때

**증상**: 보고서 미리보기 화면에 "등록된 하자가 없습니다" 메시지만 표시됨

**해결 방법:**

1. **하자 등록 확인**
   - 하자가 실제로 등록되어 있는지 확인
   - 하자 등록 화면에서 하자를 등록하세요

2. **케이스 확인**
   - 하자가 올바른 케이스에 등록되어 있는지 확인
   - 케이스 목록에서 케이스를 선택하세요

3. **데이터 동기화**
   - 페이지를 새로고침하여 최신 데이터를 불러오세요
   - 또는 보고서 미리보기를 다시 열어보세요

---

### SMS 발송이 실패할 때

**증상**: "문자 발송(모의)" 버튼을 클릭해도 반응이 없음

**해결 방법:**

1. **케이스 선택 확인**
   - 케이스가 선택되어 있는지 확인
   - "케이스를 먼저 선택해주세요" 메시지가 표시되면 케이스를 선택하세요

2. **전화번호 형식 확인**
   - 전화번호를 올바른 형식으로 입력했는지 확인
   - 예: `010-1234-5678` 또는 `01012345678`

3. **인터넷 연결 확인**
   - 인터넷 연결 상태를 확인하세요

---

### 보고서 내용이 업데이트되지 않을 때

**증상**: 새로 등록한 하자가 보고서에 표시되지 않음

**해결 방법:**

1. **페이지 새로고침**
   - 브라우저에서 페이지를 새로고침 (F5)
   - 또는 보고서 미리보기를 다시 열어보세요

2. **하자 저장 확인**
   - 하자가 정상적으로 저장되었는지 확인
   - 하자 목록 화면에서 하자가 표시되는지 확인

3. **케이스 확인**
   - 하자가 올바른 케이스에 등록되어 있는지 확인

## ❓ 자주 묻는 질문

### Q1. 보고서에 장비 점검 결과도 포함되나요?

**A:** 네, 점검원 권한을 가진 사용자의 경우 장비 점검 결과도 보고서에 포함됩니다.

- 열화상 점검
- 공기질 측정 (TVOC, HCHO, CO₂)
- 라돈 측정
- 레벨기 측정

일반 입주자는 하자 정보만 확인할 수 있습니다.

---

### Q2. 보고서를 이메일로 발송할 수 있나요?

**A:** 현재는 SMS 발송만 지원하며, 이메일 발송 기능은 향후 업데이트에서 제공될 예정입니다.

---

### Q3. PDF 다운로드는 언제 사용할 수 있나요?

**A:** PDF 다운로드 기능은 현재 개발 중이며, 향후 업데이트에서 제공될 예정입니다.

---

### Q4. 보고서를 인쇄할 수 있나요?

**A:** 현재는 브라우저의 인쇄 기능을 사용하여 보고서를 인쇄할 수 있습니다.

1. 보고서 미리보기 화면에서
2. 브라우저 메뉴 → 인쇄 (또는 Ctrl+P / Cmd+P)
3. 인쇄 설정 후 인쇄

---

### Q5. 보고서에 사진이 표시되지 않아요

**A:** 사진이 표시되지 않는 경우:

1. **사진 업로드 확인**
   - 하자 등록 시 사진이 정상적으로 업로드되었는지 확인
   - 사진이 업로드되지 않았다면 "근거리" 또는 "원거리" 텍스트만 표시됩니다

2. **인터넷 연결 확인**
   - 사진은 서버에서 불러오므로 인터넷 연결이 필요합니다
   - 인터넷 연결 상태를 확인하세요

3. **브라우저 캐시 삭제**
   - 브라우저 캐시를 삭제하고 다시 시도해보세요

---

### Q6. 보고서를 여러 번 발송할 수 있나요?

**A:** 네, 보고서는 여러 번 발송할 수 있습니다. "문자 발송(모의)" 버튼을 클릭하여 원하는 만큼 발송할 수 있습니다.

---

### Q7. 보고서 발송 기록을 확인할 수 있나요?

**A:** 현재는 보고서 발송 기록을 확인하는 기능이 없습니다. 향후 업데이트에서 제공될 예정입니다.
