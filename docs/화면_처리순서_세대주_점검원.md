## 실제 배포 앱에서 캡처하기

실제 화면 캡처가 필요하면 **Puppeteer 스크립트**를 사용할 수 있습니다.

- **스크립트**: `backend/scripts/capture-screenshots.js`
- **실행**: `BASE_URL=https://웹앱URL node scripts/capture-screenshots.js` (backend 디렉터리에서)
- **저장**: `docs/screens/` (기본값)
