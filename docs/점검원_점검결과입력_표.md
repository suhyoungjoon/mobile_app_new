# 점검원 점검결과 입력 내용 (표 정리)

## 공통

- **대상**: 하자 목록에서 선택한 **하자 1건** 단위로 입력
- **저장**: 탭별로 [저장] 시 해당 타입의 `inspection_item` 1건 생성 (동일 하자에 여러 건 추가 가능)
- **재설정**: [재설정] 시 현재 폼만 초기화

---

## 1. 공기질

| 항목 | 입력 방식 | 필수 | 비고 |
|------|-----------|------|------|
| 위치 | 텍스트 | ○ | 예: 거실, 침실 |
| 공정 | 텍스트 | - | 예: 마감, 바닥재 |
| 공정 유형 | 선택 | - | 선택 안 함 / Flush-out / Bake-out |
| TVOC | 숫자 (0~20, 소수 2자리) | - | 단위: mg/m³ |
| HCHO | 숫자 (0~20, 소수 2자리) | - | 단위: mg/m³ |
| CO2 | 숫자 (0~10000) | - | 단위: ppm |
| 메모 | 텍스트(여러 줄) | - | 공기질 측정 특이사항 |
| 결과 | 선택 | - | 정상 / 확인요망 / 해당없음 |
| 사진 | 파일 1개 | - | 📷 사진 추가 |

---

## 2. 라돈

| 항목 | 입력 방식 | 필수 | 비고 |
|------|-----------|------|------|
| 위치 | 텍스트 | ○ | 예: 침실, 거실 |
| 공정 | 텍스트 | - | 예: 마감, 바닥재 |
| 라돈 농도 | 숫자 (0~5000, 소수 2자리) | ○ | - |
| 단위 | 선택 | - | Bq/m³ / pCi/L |
| 메모 | 텍스트(여러 줄) | - | 라돈 측정 특이사항 |
| 결과 | 선택 | - | 정상 / 확인요망 / 해당없음 |
| 사진 | 파일 1개 | - | 📷 사진 추가 |

---

## 3. 레벨기

| 항목 | 입력 방식 | 필수 | 비고 |
|------|-----------|------|------|
| 위치 | 텍스트 | ○ | 예: 주방, 거실 |
| 공정 | 텍스트 | - | 예: 바닥, 타일 |
| 기준 (mm) | 숫자 (1~999, 소수 1자리) | - | 기본값 150 |
| 1번 좌 (mm) | 숫자 (±10, 소수 1자리) | - | 4 point ±10mm |
| 1번 우 (mm) | 숫자 (±10, 소수 1자리) | - | 4 point ±10mm |
| 2번 좌/우 (mm) | 숫자 (±10, 소수 1자리) | - | 동일 |
| 3번 좌/우 (mm) | 숫자 (±10, 소수 1자리) | - | 동일 |
| 4번 좌/우 (mm) | 숫자 (±10, 소수 1자리) | - | 동일 |
| 메모 | 텍스트(여러 줄) | - | 레벨기 측정 특이사항 |
| 결과 | 선택 | - | 이상없음 / 확인요망 / 해당없음 |
| 사진 | 파일 1개 | - | 📷 사진 추가 |

---

## 4. 열화상

| 항목 | 입력 방식 | 필수 | 비고 |
|------|-----------|------|------|
| 위치 | 텍스트 | ○ | 예: 거실, 침실 |
| 사진 | 파일 1개 | - | 📷 사진 추가 (열화상 이미지) |
| 점검내용 | 텍스트(여러 줄) | - | 열화상 점검 내용 |

---

## 요약 표 (탭별 입력 항목)

| 구분 | 공기질 | 라돈 | 레벨기 | 열화상 |
|------|--------|------|--------|--------|
| 위치 | ○ | ○ | ○ | ○ |
| 공정 | ○ | ○ | ○ | - |
| 측정값 | TVOC, HCHO, CO2 | 라돈 농도+단위 | 기준 mm, 1~4번 좌/우 mm | - |
| 공정 유형 | Flush-out/Bake-out | - | - | - |
| 메모 | ○ | ○ | ○ | 점검내용(동일) |
| 결과 | 정상/확인요망/해당없음 | 동일 | 이상없음/확인요망/해당없음 | - |
| 사진 | 1장 | 1장 | 1장 | 1장 |
